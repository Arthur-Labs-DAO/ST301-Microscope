# ST301 Microscope Auto-Launcher

Auto-detect and launch ST301 USB microscope with VLC on Linux.

## Quick Install

```bash
git clone https://github.com/Arthur-Labs-DAO/ST301-Microscope.git
cd ST301-Microscope
chmod +x install.sh
./install.sh
```

## Usage

After installation:

**Command Line:**

```bash
microscope
```

**Applications Menu:**
Search for "ST301 Microscope"
